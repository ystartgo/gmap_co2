安裝
- Chrome → 擴充功能 → 開發者模式 → 載入未封裝 → 指向 `chrome_ext/`

使用
- 在 Google 地圖規劃路線頁面，右下角會出現 CO2 面板。
- 面板會自動讀取距離與模式，可調整乘客數與係數。
- 按「新增到列表」可累積多筆；按「匯出Excel(CSV)」下載 `S3C6_商務旅行_Demo-2025.csv`；按「下載截圖」保存目前地圖 PNG。

設定
- 在擴充的「選項」頁面可調整各交通工具的 `kgCO2e/pkm` 係數。

說明
- 目前以 CSV 匯出；若需嵌入圖片的 XLSX，可後續加入 ExcelJS 打包將截圖以 base64 形式嵌入到 `截圖` 工作表。
